Quick Start
1. Configure token and API base URL.
2. Create a profile (runtime defaults).
3. Launch instances with model + per-instance concurrency.
4. Use copy buttons for base/chat/model routing values.

Profiles store runtime defaults; instances store effective model and inflight limits.
